Using bigbox3d.html without any parameter
Example

When you call bigbox3d.html without any parameter, it will use the default values and display a placeholder box.

Examine the template images in order to know how to save your scanned images:

  • template-front.jpg
  • template-back.jpg
  • template-top.jpg
  • template-bottom.jpg
  • template-left.jpg
  • template-right.jpg

Provide Path and Base Name of the Image Files
Example

In this example we have the images in 

/bigbox3d/img/
and their base name is 
ff2_
So the complete URL parameters we use are:
?path=/bigbox3d/img/&name=ff2_

Provide File Extension if you're not using .jpg Files
Example

In this example we have the png instead of jpg images so we use 

ext=png
The complete URL parameters for this example are:
?path=/bigbox3d/img/&name=sj_&ext=png

Change the Background Color
Example

In this example we use a black background by providing the following URL parameter: 

bg=000000
The complete URL parameters for this example are:
?path=/bigbox3d/img/&name=Sid%20Meier%27s%20Pirates%20-%20Limited%20Collector%27s%20Edition-&bg=000000
