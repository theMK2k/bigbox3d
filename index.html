Using bigbox3d.html without any parameter
Example

When you call bigbox3d.html without any parameter, it will use the default values and display a placeholder box.

Provide Path and Base Name of the Image Files
Example

In this example we have the images in 

/bigbox3d/img/
and their base name is 
ff2_
So the URL parameters we use are:
?path=/bigbox3d/img/&name=ff2_
